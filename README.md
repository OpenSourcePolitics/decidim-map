# Decidim::Map

React / Leaflet / GraphQL components for map display

## Usage

Map will be available as a menu in the main navigation.

## Installation

**need Decidim 0.15+**  

Add this line to your application's Gemfile:

```ruby
gem 'decidim-map
```

And then execute:

```bash
bundle
```
